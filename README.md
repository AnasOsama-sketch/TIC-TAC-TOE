# TIC-TAC-TOE
This is a game I've created with python and I thought it would be good to share it with you!
<hr>
Controls:
---------
Player 1 (X) : w for up,
               s for down,
               a for left,
               d for right.
               
Player 2 (O) : i for up,
               k for down,
               j for left,
               l for right.
